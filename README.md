## 🚀 התקנה והפעלה

### ⚡ הפעלה מהירה — Windows (מומלץ)

**פעם ראשונה:**

1. **הורד את הפרויקט** — לחץ על הכפתור הירוק `<> Code` ← `Download ZIP`
2. **חלץ** את ה-ZIP לתיקייה כלשהי (למשל `C:\kids-chores\`)
3. **לחץ פעמיים** על הקובץ `install.bat`
4. המערכת תתקין אוטומטית ותיפתח בדפדפן 🎉

**הפעלות הבאות:**

לחץ פעמיים על `launch.vbs` — נפתח ישירות ללא חלון שחור

---

### 🐍 הפעלה ידנית (כל מערכת הפעלה)

#### דרישות
- **Python 3.8 ומעלה** — [הורדה מ-python.org](https://www.python.org/downloads/)
  - בהתקנה: ✅ סמן **"Add Python to PATH"**

#### שלבים

```bash
# 1. שכפל את הפרויקט (או הורד ZIP)
git clone https://github.com/uziel110/Kids-Chores-Manager.git
cd Kids-Chores-Manager

# 2. התקן חבילות (פעם אחת בלבד)
pip install -r requirements.txt

# 3. הפעל
python main.py
```

פתח דפדפן וכנס ל: **http://localhost:8000**

---

### 📱 גישה מטלפון / טאבלט ברשת הביתית

```
http://[כתובת-IP-של-המחשב]:8000
```

**איך למצוא את כתובת ה-IP?**
- Windows: פתח Command Prompt ← הקלד `ipconfig` ← חפש `IPv4 Address`
- לדוגמה: `http://192.168.1.100:8000`

> 💡 **טיפ:** שמור את הכתובת כסימניה בנייד לגישה נוחה

## 📖 לימוד משניות בעל פה

פיצ'ר ייחודי ללימוד תורני — ניהול מלא של שינון משניות עם מעקב ואישורים.

### Flow
```
הורה: פאנל → "📖 משניות" → "הוסף משימת משניות" → בחר ילד + נקודות למשנה
                                        ↓
ילד: מסך מטלות → "📖 משניות מיוחדות" → בחר סדר → מסכת → פרק → ✓ משניות
                                        ↓
                              לוחץ "📚 אני לוקח ללמוד!"
                                        ↓
                         לומד ← לוחץ "✋ למדתי!" על כל משנה
                                        ↓
         הורה: אשר ✔  /  ללמוד שוב 🔄  /  החזר לחלוטין ↩
                                        ↓
                            ✅ נקודות נוספות אוטומטית
```

**כלל:** כל ילד יכול לקחת כל משנה פעם אחת בלבד — אלא אם ההורה החזיר אותה.

**כיסוי:** ששה סדרי משנה · 63 מסכתות · כל הפרקים עם מניין משניות מדויק

## 🗂️ מבנה הפרויקט

```
Kids-Chores-Manager/
├── main.py                  # שרת FastAPI — כל ה-API
├── requirements.txt         # חבילות Python
├── install.bat              # התקנה + הפעלה אוטומטית (Windows)
├── launch.vbs               # הפעלה שקטה ללא חלון שחור
└── static/
    ├── index.html           # SPA — ממשק המשתמש המלא
    ├── css/style.css        # עיצוב RTL, mobile-first
    ├── screenshots/         # צילומי מסך (README)
    └── js/
        ├── state.js         # state גלובלי, api(), toast()
        ├── ui.js            # שומר מסך, טיימרים, קונפטי
        ├── home.js          # מסך ראשי + לוח מובילים
        ├── chores.js        # מסך מטלות לילד
        ├── rewards.js       # חנות פרסים
        ├── stats.js         # סטטיסטיקות
        ├── parent.js        # פאנל הורים
        ├── mishna_data.js   # נתוני משנה (6 סדרים, 63 מסכתות)
        ├── mishna.js        # לוגיקת לימוד משניות
        ├── onboarding.js    # מדריך ראשוני
        └── init.js          # אתחול
```
