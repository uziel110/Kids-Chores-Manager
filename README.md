# 🌟 מערכת מטלות לילדים

מערכת ניהול מטלות לילדים לשימוש מקומי בבית

---

## 🚀 הפעלה מהירה

### דרישות מינימום
- Python 3.8+
- חיבור אינטרנט (רק לפעם ראשונה, להורדת חבילות)

### הפעלה

```bash
python start.py
```

המערכת תיפתח אוטומטית בדפדפן בכתובת: http://localhost:8000

---

## 📁 מבנה הפרויקט

```
kids-chores/
├── start.py          # קובץ הפעלה ראשי
├── main.py           # שרת FastAPI
├── data.json         # מסד הנתונים (JSON)
├── requirements.txt  # חבילות Python
├── README.md         # קובץ זה
└── static/
    └── index.html    # ממשק משתמש מלא
```

## 👨‍👩‍👧 הוראות שימוש

### ילדים
1. לחץ על הכרטיס שלך
2. בחר מטלה מהרשימה (או לחץ "תן לי מטלה!" למטלה אקראית)
3. בצע את המטלה
4. לחץ "סיימתי!" בלוח הפעיל

### הורים
1. לחץ "הורים" בסרגל העליון
2. הכנס סיסמה (ברירת מחדל: **1234**)
3. אשר/דחה מטלות שהושלמו
4. נהל ילדים, מטלות ופרסים
