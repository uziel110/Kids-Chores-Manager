## 🚀 התקנה — 4 צעדים פשוטים

**1. הורד את התוכנה**
לחץ על הכפתור הירוק **`<> Code`** בראש הדף ← **`Download ZIP`** ← חלץ לתיקייה נוחה

**2. התקן Python**
[לחץ כאן להורדה](https://www.python.org/downloads/) ← בהתקנה חשוב: ✅ סמן את **"Add Python to PATH"**

**3. הפעל לראשונה**
לחץ פעמיים על הקובץ **`install.bat`** — הכל מותקן ונפתח לבד 🎉

**4. הפעלות הבאות**
לחץ פעמיים על **`launch.vbs`**

---

### 📱 רוצה גישה גם מהטלפון?

כשהמחשב פועל, פתח בטלפון את הדפדפן והקלד:
**`http://[כתובת-IP-של-המחשב]:8000`**

כדי לדעת מה ה-IP: לחץ **Win+R** ← הקלד `cmd` ← הקלד `ipconfig` ← חפש את `IPv4 Address`

> 💡 שמור את הכתובת כסימניה בנייד לגישה נוחה בפעמים הבאות
