# 🌟 מערכת מטלות לילדים — Kids Chores Manager

> מערכת ניהול מטלות, נקודות ופרסים לילדים — לשימוש מקומי בבית  
> ממשק עברי מלא, RTL, מתאים לנייד ולטאבלט

---

## ✨ תכונות עיקריות

### 👧👦 לילדים
- בחירת ילד מלוח ראשי עם אווטר / תמונה אישית
- רשימת מטלות זמינות עם מיקום, נקודות וטיימר
- כפתור **"תן לי מטלה!"** — בחירה אקראית
- מעקב מטלות פעילות עם ספירת זמן לאחור
- **📖 לימוד משניות בעל פה** — browser לבחירת סדר / מסכת / פרק / משניות
- חנות פרסים — פדיון נקודות
- 🎉 קונפטי + אנימציות בכל אישור

### 👨‍👩‍👧 להורים (מוגן סיסמה)
- **אישורים** — אשר/דחה מטלות, אשר פרסים, אשר משניות
- **ניהול מטלות** — CRUD מלא: נקודות, מיקום, חלון זמן, חזרתיות, גיל, הגבלת לוקחים
- **ניהול ילדים** — הוסף/ערוך, תמונה/אמוג'י, צבע, גיל, מין
- **ניהול פרסים** — תמונה/אמוג'י, עלות נקודות, מגבלת מימוש
- **📖 משימות משניות** — צור משימה לילד עם נקודות למשנה
- **סטטיסטיקות** — לוח מובילים שבועי, פעילות יומית, היסטוריית נקודות
- **גיבוי/שחזור** — ייצוא/ייבוא JSON מלא

## 📖 מערכת לימוד משניות בעל פה (חדש!)

### Flow ללמידה
```
הורה יוצר משימה לילד (+ נקודות למשנה)
    ↓
ילד עובר לטאב "📖 משניות מיוחדות"
    ↓
בוחר: סדר → מסכת → פרק → ✓ משניות
    ↓
לוחץ "📚 אני לוקח ללמוד!"
    ↓
לומד, ואז לוחץ "✋ למדתי!" על כל משנה
    ↓
הורה מאשר ✔ / מחזיר לחזרה 🔄 / מחזיר לחלוטין ↩
    ↓
נקודות נוספות אוטומטית ✅
```

**כלל:** כל ילד יכול לקחת כל משנה פעם אחת בלבד — אלא אם ההורה החזיר אותה.

**הכיסוי:** ששה סדרי משנה · 63 מסכתות · כל הפרקים

## 🚀 הפעלה מהירה

### דרישות
- Python 3.8+
- חיבור אינטרנט לפעם ראשונה בלבד (להורדת חבילות)

### Windows — הפעלה בלחיצה אחת
```
לחץ פעמיים על: install.bat
```

### הפעלה ידנית
```bash
pip install -r requirements.txt
python main.py
```

המערכת נפתחת אוטומטית על: **http://localhost:8000**

## 🗂️ מבנה הפרויקט

```

├── main.py                  # שרת FastAPI — כל ה-API endpoints

        ├── chores.js        # מסך מטלות לילד
        ├── rewards.js       # חנות פרסים
        ├── stats.js         # סטטיסטיקות

        ├── mishna.js        # לוגיקת לימוד משניות
        ├── onboarding.js    # מדריך ראשוני
        └── init.js          # אתחול האפליקציה
```
